관리자 대시보드(/admin)를 아래 명세대로 전면 재구성해줘. 모호성 판단해서 높으면 딥 인터뷰 진행하고, 명확하면 바로 Ralph 모드로 끝까지 개발해줘. 수락 기준 다 통과할 때까지 중간에 나 안 부르고 알아서 반복해서 마무리하고, 로컬 빌드까지 하고 프로덕션 배포한 다음 결과만 알려줘. DB 스키마 변경은 없어야 하고(기존 테이블만), 혹시 필요하면 마이그레이션 SQL 초안만 만들고 실행하지 마.

━━ 핵심 컨셉 ━━
사장이 관리자 페이지에 딱 들어오면 "회사 전체 손익 현황"이 먼저 보이고, 큰 그림에서 비용/가입 항목을 클릭해 점점 상세하게 파고드는(drill-down) 구조. 기본 정산 기간은 "이번 달"(요금제가 월 구독이라 매출·비용 모두 월 기준으로 손익 일치). 기간 필터로 오늘/최근7일/이번달 전환 가능.

━━ 진입 구조 수정 ━━
지금 /admin 첫 화면이 "대화 내역" 탭 + "아이 선택하세요" 드롭다운으로 시작하는데, 이걸 뒤집어줘. /admin 진입 시 아이 선택 없이 바로 "회사 전체 현황 대시보드"가 랜딩으로 떠야 함. 상단 "아이 선택" 드롭다운은 첫 화면에서 제거.

━━ 1단계: 첫 화면 = 회사 전체 손익 (이번 달 기본) ━━
최상단에 제일 크게 3개 덩어리:
1) 들어올 돈 (예상 매출) = 가입 사용자 요금제 합계. 현재 무료 베타라 "전원 유료 가정" 명시.
2) 나갈 돈 (총 비용) = AI 비용(STT+TTS+라이브+LLM) + 인프라 고정비(Vercel+Supabase).
3) 남는 돈 (순이익) = 매출-비용. 흑자 초록/적자 빨강, 제일 크게 강조.
각 숫자에 직전 동기간 대비 증감률(%).
그 아래 보조: 총 가입 고객(아이) 수, 요금제별 인원(Care Start N명/Care Insight N명/Care Premium N명), 대화 세션 수.
그 아래 일별 손익 추이 그래프(recharts): 매출선·비용선 겹치기. 데이터 없으면 0으로 채운 빈 그래프(빈 화면 금지).

━━ 2단계: 클릭해서 파고들기 ━━
[나갈 돈 클릭] → 비용을 항목별로 분해, 비용 큰 순 정렬:
  - AI 비용 4종: STT / TTS / Gemini 라이브 / LLM(제미나이 텍스트) — 각 사용량, 우리 추정, 실제 청구액, 전체 대비 %.
  - 인프라 고정비: Vercel, Supabase.
  각 AI 서비스 항목 클릭 → 그 서비스 제일 많이 쓴 탑 유저 순위(탑10, 아이 이름 + 사용량 + 비용). 4종 다 통일된 방식.
[가입자/매출 클릭] → 티어별 인원 분포(Care Start/Insight/Premium) → 특정 티어 클릭 → 그 티어에 속한 사용자 명단(이름, 가입일, 월요금, 이번 기간 원가, 마진).

━━ 3단계: 개별 사용자 상세 ━━
사용자(아이) 클릭 → 그 아이 하나 상세: 대화 세션 수, 서비스별 사용량·비용, 기간별 추이, 대화 내역, 안전 이벤트.

━━ 비용 데이터 출처 ━━
- STT, TTS: GCP 클라우드 빌링(BigQuery billing_export, K-Bestie3 프로젝트) 실제 청구액. 이미 연동됨. 데이터 반영 24~48시간 지연되니 최근 값은 "집계 중" 표시.
- Gemini 라이브 + LLM(텍스트): 구글 AI 스튜디오라 자동 export 없음. usage_events에 쌓인 토큰/시간 사용량 × 단가로 "계산". 화면엔 계산값 표시하고 "AI Studio 월말 대조" 표기. 단가는 pricing.ts 상수로.
  ※ 확인 필요: 현재 텍스트 모델이 Gemini 2.5 Flash와 3.1 Flash Lite 두 종이 섞여 쓰이고 있음. 코드에서 어느 상황에 어느 모델 쓰는지, usage_events에 모델명이 기록되는지 확인해서 모델별 단가를 정확히 반영해줘. 안 되면 대표적인 모델 단가로 근사 후 주석 표시.
- 인프라 고정비 상수(pricing.ts): Vercel 월 30,000원, Supabase 월 37,500원(각 $20/$25 × 환율 1,500원 근사치, 실제 청구서 나오면 갱신). 매달 고정 반영.

━━ 요금제 단가(pricing.ts) ━━
Care Start 9,900원/월, Care Insight 14,900원/월, Care Premium 150,000원/월.

━━ 공통 ━━
- 지금은 실제 데이터가 거의 0(GCP 어제 켜서 아직 0원, 사용량 로그 이제 쌓이는 중). 데이터 없어도 화면은 0으로 채워 정상 렌더링(빈 화면/에러 금지). 데이터 들어오면 자동 반영되게.
- 기존 --hb-* 디자인 토큰 유지. 손익 3개 숫자 최상단·최대 크기, 나머지 아래. 실제 모니터링 도구(Langfuse/Helicone/GCP 빌링) 수준의 정돈된 카드·표·그래프 레이아웃.
- 아이 이름/ID 매칭 로직 전체 점검(이전에 선택값이 다른 아이로 표시되던 혼동 방지).
