[부모 홈 대시보드 교체]
- 부모 홈을 첨부 이미지의 '아이 현황 보기' 8개 카드 구조로 교체한다.
- 기존 부모 홈 컴포넌트를 이 구조로 업데이트한다 (확장/수정 방식).
- 새 파일/컴포넌트를 만들게 되면 기존 부모 홈 관련 코드는 반드시 제거한다 (중복 금지).
- app/demo 는 절대 건드리지 않는다 (더미 데이터는 데모에만 남긴다).
- 실제 부모 홈은 더미 텍스트를 쓰지 않고 전부 리얼 데이터만 표시한다.

[데이터 소스 - 8개 카드 전부 동일 원칙]
- 모든 카드 내용은 AI 대화 로그 분석 기반이다.
- 04시 일일 배치가 대화 로그를 분석해 일일 리포트를 생성하면서,
  같은 배치에서 대시보드 8개 카드 내용도 함께 채워/갱신한다.
- 갱신 주기: 하루 1회 04시 (실시간 아님).
- 감정 힌트 카드의 3단계 판정(daily_reports.emotion_level)도 이 배치의 일부다.

[3번 감정 힌트 카드 = 3단계 구조]
- 1단계: 라벨 "안정적이에요" / 초록 / 🌿 / "특별한 걱정 신호 없이 잘 지내고 있어요."
- 2단계: 라벨 "마음 살펴주세요" / 노랑·주황 / 🍂 / "속상하거나 힘든 일이 있는 것 같아요. 오늘 저녁에 따뜻하게 얘기 나눠보세요."
- 3단계: 라벨 "지금 함께해주세요" / 빨강 / ❤️‍🩹 / "아이가 많이 힘들어하는 신호가 보여요. 지금 곁에서 살펴봐 주시고, 필요하면 전문기관 안내를 참고하세요."
